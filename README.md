# Movie Sentiment Analysis

Проект для анализа тональности текстовых рецензий на фильмы. Включает API-сервис на FastAPI, веб-интерфейс на Streamlit, интеграцию с MLflow для отслеживания экспериментов и CI/CD пайплайн.

## Описание проекта

Сервис принимает текстовую рецензию на фильм и возвращает предсказание тональности: положительная, отрицательная или нейтральная. Для анализа используется обученная модель машинного обучения с дополнительной мета-признаковой инженерией (длина текста, количество восклицательных и вопросительных знаков, количество заглавных букв).

## Функциональность

- REST API для предсказания тональности рецензий
- Веб-интерфейс на Streamlit для удобного взаимодействия
- Валидация входных текстов (минимальная длина, наличие букв, максимальная длина)
- Отображение уверенности модели в предсказании
- Вывод 5 наиболее влиятельных слов для данного текста (в Streamlit)
- Отслеживание экспериментов через MLflow
- CI/CD пайплайн (GitHub Actions)
- Контейнеризация через Docker

## Технологический стек

- **Python 3.11**
- **FastAPI** — создание REST API
- **Streamlit** — веб-интерфейс
- **Scikit-learn** — обучение и использование модели
- **CatBoost** — базовый классификатор (используется в составе пайплайна)
- **MLflow** — отслеживание экспериментов и логирование моделей
- **Joblib** — сериализация модели
- **Pytest** — тестирование
- **Docker** — контейнеризация
- **GitHub Actions** — CI/CD

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/katherinza/movie-sentiment.git
cd movie-sentiment
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Загрузка данных для обучения (опционально)

Для обучения модели необходимы данные IMDb. Если вы планируете переобучать модель или запускать ноутбуки с обучением, загрузите данные одним из способов.

#### Способ 1 — через HuggingFace datasets

```bash
pip install datasets
python -c "from datasets import load_dataset; ds = load_dataset('imdb'); ds.save_to_disk('data/imdb')"
```

После выполнения команды данные сохранятся в папку `data/imdb/`. Для использования в скриптах обучения потребуется изменить пути загрузки на эту папку или использовать оригинальную структуру `aclImdb`.

#### Способ 2 — прямая загрузка архива со Stanford

```bash
wget https://ai.stanford.edu/~amaas/data/sentiment/aclImdb_v1.tar.gz
tar -xzf aclImdb_v1.tar.gz
```

После распаковки появится папка `aclImdb/` с подпапками `train/` и `test/`. В коде обучения используется именно эта структура (по умолчанию `data/aclImdb`).

> Примечание: для работы уже обученной модели (API и Streamlit) данные не требуются — модель загружается из файла `model_meta.joblib`. Данные нужны только для повторного обучения или экспериментов.

### 4. Запуск FastAPI сервера

```bash
uvicorn app.main:app --reload
```

Сервер будет доступен по адресу: `http://localhost:8000`

Документация API доступна по адресу: `http://localhost:8000/docs`

### 5. Запуск MLflow UI (опционально)

Для просмотра логов экспериментов:

```bash
set MLFLOW_ALLOW_FILE_STORE=true
mlflow ui --host 0.0.0.0 --port 5000
```

MLflow UI будет доступен по адресу: `http://localhost:5000`

### 6. Запуск Streamlit интерфейса

```bash
streamlit run streamlit_app.py
```

Streamlit приложение будет доступно по адресу: `http://localhost:8501`

### 7. Запуск тестов

```bash
pytest tests/
```

### 8. Запуск через Docker

```bash
docker build -t movie-sentiment .
docker run -p 8000:8000 movie-sentiment
```

## API Endpoints

### GET /health

Проверка работоспособности сервиса.

**Ответ:**
```json
{
  "status": "ok"
}
```

### POST /predict

Предсказание тональности рецензии.

**Тело запроса:**
```json
{
  "review": "This movie was amazing, the acting was brilliant..."
}
```

**Ответ:**
```json
{
  "sentiment": "positive",
  "confidence": 0.95,
  "warning": null
}
```

**Возможные значения sentiment:** `positive`, `negative`, `neutral`

**Валидация текста:**
- Минимальная длина: 10 символов
- Максимальная длина: 10 000 символов
- Текст должен содержать буквы латинского алфавита

## Обучение модели

Модель обучалась на датасете рецензий к фильмам с использованием пайплайна, включающего:

1. TF-IDF векторизацию текста
2. Мета-признаки (длина текста, количество восклицательных и вопросительных знаков, количество заглавных букв)
3. CatBoost классификатор

Результаты экспериментов логируются в MLflow.

## CI/CD

Проект настроен на автоматическое тестирование и деплой через GitHub Actions при пуше в ветку `main`.
